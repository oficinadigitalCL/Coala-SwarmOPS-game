## 🏛️ El Templo de Thot

Adéntrate en el **Templo de la Sabiduría Digital**, donde **Thot** —dios egipcio de la sabiduría, la escritura y la ciencia— te guiará a través de 3 niveles para convertirte en un **Arquitecto de Enjambres**.

Aprende cómo los agentes koala 🐨 colaboran en el Árbol Sagrado de COALA para construir software como un enjambre inteligente. Cada nivel te enseña un concepto clave de la automatización con agentes IA.

> ⚡ "Después de que aprendas COALA-SwarmOps entenderás mucho sobre automatización y memoria cognitiva"

## 🧠 ¿Qué aprenderás?

- **Nivel 1 — El Árbol Sagrado:** Los 6 roles de un swarm de agentes y sus responsabilidades
- **Nivel 2 — Sabiduría de Thot:** Conceptos de automatización, significado de COALA, filosofía de enjambres
- **Nivel 3 — El Flujo Sagrado:** El pipeline COALA paso a paso y por qué el orden importa

## 🚀 Desarrollo Local

```bash
# Clona el repo
git clone https://github.com/oficinadigitalcl/Coala-SwarmOPS-game.git

# Abre index.html en tu navegador
# No necesita servidor, es un archivo HTML autocontenido
```

O simplemente abre [`index.html`](index.html) directamente — es un **single file** sin dependencias externas.

## 🛠️ Tecnología

- **Vanilla HTML + CSS + JavaScript** — Sin frameworks, sin build tools
- **CSS Custom Properties** — Paleta egipcia moderna
- **Web Audio API** — Sonidos procedurales (tonos sintetizados)
- **localStorage** — Persistencia de puntajes altos
- **Responsive** — Funciona en mobile, tablet y desktop
- **Single File** — Todo inline: CSS en `<style>`, JS en `<script>`
